MedCase Agent — System Architecture

Autonomous clinical reasoning tutor for medical students · Next.js + FastAPI + Gemini + MCP · Google for Startups AI Agents Challenge

This tool is for medical education and simulated clinical reasoning practice only. It does not provide real patient diagnosis or treatment.

API surface — FastAPI routes

MethodRouteRouterPurpose
GET/healthmainService health check
GET/topicsmainAvailable topics & difficulties
POST/session/startsessionsCreate session, generate opening case
POST/session/{id}/answersessionsEvaluate answer, return feedback + next question + vitals
GET/session/{id}/summarysessionsFinal score, strong/weak topics, recommendations
GET/session/{id}/resumesessionsRestore an interrupted active session
GET/student/{id}/dashboardstudentsSessions + stats + weak topics in one call
GET/student/{id}/weak-topicsstudentsAggregated weak topics with frequency
GET/student/{id}/sessionsstudentsSession history, newest first
POST/GET/osce/*osceTimed OSCE station flow + blended scoring
